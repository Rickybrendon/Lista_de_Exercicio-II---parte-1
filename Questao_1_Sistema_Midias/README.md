# Questão 1 – Sistema de Mídias Educacionais

## Descrição
Sistema que utiliza uma classe abstrata para representar mídias educacionais,
demonstrando herança e polimorfismo.

## Conceitos aplicados
- Classe Abstrata (ABC)
- Herança
- Polimorfismo

## Perguntas teóricas

**Qual é a classe base do sistema?**  
Midia.

**Onde ocorre polimorfismo?**  
No método `reproduzir()`, que possui implementações diferentes em cada subclasse.

**Por que usar uma classe abstrata?**  
Para garantir que toda mídia implemente os comportamentos obrigatórios.

```bash
python main.py
