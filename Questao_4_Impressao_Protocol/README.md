# Questão 4 – Sistema de Impressão com Protocol

## Descrição
Sistema que utiliza Protocol para definir um contrato estrutural,
sem exigir herança explícita.

## Perguntas teóricas

**Onde está o contrato nesse caso?**  
No protocolo `Imprimivel`.

**Por que as classes não precisam herdar do protocolo?**  
Porque o Python usa tipagem estrutural.

**Esse caso se aproxima mais de ABC ou duck typing?**  
Duck typing formalizado.

**Diferença para a Questão 3?**  
Na Questão 3 o contrato é por herança, aqui é estrutural.
